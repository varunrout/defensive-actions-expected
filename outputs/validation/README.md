# Historical Validation Outputs

The existing files under `outputs/validation/` are historical artifacts. They were generated before the methodology corrections for event context, target wiring, visibility geometry, freeze-frame role features, and model feature specifications.

These outputs must not be treated as current model results, corrected coefficients, updated feature-selection decisions, or valid post-correction metrics.

Corrected validation outputs should only be created after the full pipeline is rerun on the corrected methodology and the models are retrained from those regenerated datasets.
