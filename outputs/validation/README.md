# Historical validation outputs

The files in this directory are historical artifacts generated before the current methodology corrections.

They were produced with previous target definitions, feature names, model specifications and validation assumptions. They must not be treated as current results for the corrected pipeline.

Do not mechanically rename target names, feature names, coefficients or model metrics inside these artifacts. Renaming historical output text does not make those outputs valid under the corrected methodology.

Corrected validation outputs should only be created after the full raw-to-processed pipeline is rerun, player features are rebuilt with the corrected targets and visibility-aware features, and the models are retrained and revalidated.
